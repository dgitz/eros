This Level is a higher form of information and does not imply that anything is wrong.

WARN 

This Level implies that a program is not running as expected, but may continue to operate in a diminished capacity.

ERROR 

This Level implies that a program will not initialize or some other kind of crash.

FATAL 

This Level implies that a program has failed so bad it can cause injury to itself or others.
